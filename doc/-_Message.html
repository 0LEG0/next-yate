Class: _Message

_Message(name, broadcast, params) → {Message}

new _Message(name, broadcast, params) → {Message}

_Message is part of compatible API. Unlike Message class of core API it has static methods: install, watch, uninstall, unwatch and own methods: enqueue and dispatch.
