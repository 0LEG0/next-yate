Class: Message

Message(name, broadcast, params)

Message class is part of core API. Unlike _Message class of compatible API does not contain static methods install/uninstall/watch/unwatch and methods enqueue and dispatch.

Constructor

new Message(name, broadcast, params)

Parameters:
Name Type Description
name string message name (required)
broadcast boolean (not used)
params Object message parameters (optional, for example {id: "sip/123", caller: "12345", called: "67890"})

Example
const {Yate, Message} = require("libyate.js");
let yate = new Yate();
let m = new Message("call.drop", { id: "sip/123", reason: "timeout" });
yate.enqueue(m);
