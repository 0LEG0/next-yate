Yate(options)

Yate object provides connection to Yate's external module.

Constructor

new Yate(options)

Parameters:
Name Type Description
options Object (optional)
Properties
Name Type Description
host string address of listerning Yate's extmodule for ex. "127.0.0.1", default "undefined", stdin/stdout connected
port number port default 5040
path string socket path, default "undefined"
trackname string track name of connected script, default "next-yate"
reconnect boolean reconnect on collisions, default true
reconnnect_timeout number reconnect tries interval in milliseconds, default 10000
dispatch_timeout number auto drop the response waiting in Yate.dispatch method after the timeout in milliseconds if Engine not respond, default 10000
acknowledge_timeout number auto reply to incoming message as is if the callback function did not responded in the timeout, so as not to overload the Engine queue and not cause the engine to crash. default 10000
bufsize number sets the maximum size of transferred data in extmodule, oversize will be truncated to the specified value, default 8192.
channel boolean to run the Application in channel mode ^NNN=extmodule/nodata/node.sh example.js
