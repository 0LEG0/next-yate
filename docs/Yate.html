Class: Yate

Yate(options)

new Yate(options)

Core API Yate class. It provides connection to Yate's external module.
Parameters:
Name Type Description
options Object (optional)
Properties
Name Type Description
host string Address of listerning Yate's extmodule (for example "127.0.0.1")
port number (default 5040)
path string Socket path (default undefined)
trackname string Sets the track name of your script. (required, default value "next-yate")
reconnect boolean Try to reconnect in collisions. (default true)
reconnnect_timeout number Reconnect tries interval in milliseconds. (default 10000)
dispatch_timeout number Drops the response from Yate after dispatch_timeout in milliseconds, so that your code does not hang waiting endlessly for an answer. (default 3000)
acknowledge_timeout number Reply to the message without processing after a specified time, so as not to overload the Yate queue and not cause the engine to crash. (default 3000)
bufsize number Sets the maximum size of transferred query data in extmodule. (default 8190).

install(callback, name, priority, filter, fvalue)

Sets the handler to a specific message. The yate engine will wait for a response to its message.
Parameters:
Name Type Description
callback function function to be message handler (required)
name string message name (required)
priority number priority of the handler, as low value as high proirity (optional, default 100)
filter string set the filter to message parameter (optional, for example "called")
fvalue string filter value (optional, for example "^9999.*")
Source:
  • libyate.js, line 659
See:
  • YateMessage
  • https://docs.yate.ro/wiki/Standard_Messages
Example
const {Yate, YateMessage} = require("next-yate");

function onRoute(message) {
    message.retValue("tone/ring"); // send the incoming call to tone/ring module
}

let yate = new Yate();
yate.init();
yate.install(onRoute, "call.route", 90, "called", "^9999.*"); // set the handler to "call.route" messages where parameter message.called begins with "9999", handler has priority 90
